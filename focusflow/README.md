# FocusFlow

A minimalist productivity app that embraces constraints instead of endless features. Set a single intention, work within a defined time boundary, with gentle haptic breathing reminders.

## Philosophy

Inspired by the wired headphone trend - sometimes limitations create better experiences than infinite options. FocusFlow strips away the complexity of modern productivity apps and focuses on what matters: clear intention, bounded time, and mindful presence.

## Features

- **Single Intention**: Focus on one thing at a time
- **Time Boundary**: Set a specific duration for your focus session
- **Breathing Reminders**: Gentle visual and haptic cues to maintain mindful awareness
- **Minimalist Design**: No distractions, just what you need
- **Constraint-Based**: Limited options create better focus

## How to Use

1. Open `index.html` in your web browser
2. Set your single intention for the session
3. Define your time boundary (minutes and seconds)
4. Click "Begin" to start your focus session
5. Follow the breathing reminders to stay present
6. Complete your session mindfully

## Technical Notes

- Built as a single HTML file with inline CSS and JavaScript
- Uses the Vibration API for haptic feedback on supported devices
- Responsive design works on desktop and mobile
- No external dependencies beyond Google Fonts
- Offline-ready once loaded

The app embodies the principle that constraints can enhance rather than limit creativity and productivity.
