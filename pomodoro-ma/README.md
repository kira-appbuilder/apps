# PomodoroMA

A focus timer that embraces the Japanese concept of 間 (ma) - the meaningful pause between work sessions, featuring breathing exercises and reflection prompts.

## Features

- **Traditional Pomodoro Timer**: 25-minute work sessions with 5-minute breaks
- **間 (Ma) Breaks**: Meaningful pauses with guided breathing exercises and reflection prompts
- **Bilingual Support**: Switch between English and Japanese (日本語)
- **Progress Tracking**: Daily sessions, total sessions, and streak counter
- **Customizable Durations**: Adjust work and break periods to your preference
- **Mindful Design**: Dark, minimalist interface following KIRA design principles
- **Breathing Guidance**: Visual breathing circle with rotating instructions during breaks
- **Reflection Prompts**: Random mindfulness and reflection questions during breaks
- **Persistent Stats**: Progress saved in local storage

## Usage

1. Open `index.html` in your web browser
2. Adjust work and break durations in the settings if desired
3. Click "Start" to begin a work session
4. When the work session ends, enjoy the 間 (ma) break with breathing exercises and reflection
5. Continue alternating between work and meaningful pauses

## The Concept of 間 (Ma)

間 (ma) is a Japanese concept representing the purposeful use of pause, silence, or empty space. In PomodoroMA, breaks aren't just rest periods - they're meaningful moments for breathing, reflection, and centering yourself before the next focused work session.

## Technical Details

- Single HTML file with embedded CSS and JavaScript
- No external dependencies
- Responsive design for mobile and desktop
- Local storage for settings and progress tracking
- Built with KIRA design system
