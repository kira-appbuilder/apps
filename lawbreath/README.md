# LawBreath - Gentle Legal Document Companion

LawBreath transforms overwhelming legal documents and contracts into digestible, human-readable sections with guided breathing breaks between complex clauses.

## Features

- **Plain Language Translation**: AI-powered summaries convert legalese into everyday language
- **Gentle Pacing**: Guided breathing exercises between complex sections help maintain clarity
- **Progress Visualization**: Wave-like progress tracking shows your journey through the document
- **Complexity Assessment**: Each section is marked with complexity levels (low, medium, high)
- **Multi-language Support**: Available in English and Japanese
- **Document Import**: Upload PDF, DOC, or TXT files, or paste text directly

## How to Use

1. **Upload or Paste**: Add your legal document by uploading a file or pasting text
2. **Gentle Analysis**: The app breaks down your document into manageable sections
3. **Section Review**: Read through each section with plain language explanations and key points
4. **Breathing Breaks**: Take optional breathing exercises between complex clauses
5. **Progress Tracking**: Monitor your progress with gentle wave visualizations

## Target Users

Perfect for anyone who:
- Deals with contracts and legal documents
- Gets overwhelmed by dense legal language
- Wants to understand legal documents at their own pace
- Prefers human-centered approaches over lawyer-focused tools

## Technology

- Single HTML file with embedded CSS and JavaScript
- Responsive design following KIRA design system
- Works offline after initial load
- No external dependencies

## Getting Started

Simply open `index.html` in any modern web browser. The app works immediately without any installation or setup.
