# ContextWeaver

A personal knowledge companion that helps you have long-form conversations with your own scattered thoughts, notes, and digital fragments. Like having tea with your past self across months of thinking.

## Features

- **Fragment Capture**: Save thoughts, notes, and insights as knowledge fragments
- **Intelligent Connections**: Automatically finds connections between your ideas
- **Conversational Interface**: Chat with your accumulated knowledge using natural language
- **Temporal Context**: See how your thinking evolves over time
- **Bilingual Support**: Full English and Japanese language support
- **Local Storage**: All your data stays private on your device

## Usage

1. **Add Fragments**: Capture any thought, insight, or note in the "Add Fragment" section
2. **Start Conversations**: Ask questions about your knowledge in the conversation area
3. **Discover Connections**: Watch as the app finds and highlights connections between your ideas
4. **Track Growth**: Monitor your fragment count, conversations, and connections in the stats

## Design Philosophy

ContextWeaver follows the KIRA design system, featuring:
- Dark, contemplative interface optimized for deep thinking
- Sage green accent colors representing growth and organic connections
- Elegant typography mixing Cormorant Garamond and Space Mono fonts
- Subtle animations and micro-interactions
- Bilingual UI with seamless language switching

## Technical Details

- Single HTML file with embedded CSS and JavaScript
- Uses browser localStorage for data persistence
- Responsive design works on desktop and mobile
- No external dependencies or server required
- Includes error reporting and feedback collection

## Getting Started

Simply open `index.html` in any modern web browser. No installation or setup required.
