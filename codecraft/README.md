# CodeCraft

A minimal code snippet organizer with beautiful syntax highlighting and tag-based search for developers who want to curate their knowledge.

## Features

- **Beautiful UI**: Dark theme with elegant syntax highlighting following KIRA design principles
- **Tag-based Organization**: Add tags to your snippets and search by them
- **Multiple Languages**: Support for JavaScript, Python, HTML, CSS, Bash, SQL, JSON, Markdown, and more
- **Local Storage**: All your snippets are saved locally in your browser
- **Search & Filter**: Quickly find snippets by title, content, language, or tags
- **Edit & Delete**: Full CRUD operations for managing your code collection
- **Multi-language Support**: Interface available in English and Japanese
- **Mobile Responsive**: Works perfectly on all device sizes

## Usage

1. Open `index.html` in your browser
2. Add your first code snippet using the form at the top
3. Organize with tags for easy retrieval
4. Use the search bar to filter your collection
5. Click on tags to filter by specific topics
6. Edit or delete snippets as needed

## Language Support

CodeCraft provides syntax highlighting for:
- JavaScript
- Python
- HTML
- CSS
- Bash/Shell
- SQL
- JSON
- Markdown
- And more...

## Data Storage

All snippets are stored locally in your browser's localStorage. No server required, no data leaves your device.
