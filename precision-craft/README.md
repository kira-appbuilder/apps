# PrecisionCraft

A minimalist tool for makers and engineers that helps calculate and visualize manufacturing tolerances, inspired by LEGO's legendary 0.002mm precision standard.

## Features

- **Bilateral Tolerance Calculator**: Calculate and visualize tolerance ranges with real-time updates
- **Tolerance Stack-up Analysis**: Use Root Sum Square (RSS) method to calculate cumulative tolerances
- **Precision Grade Assessment**: Compare your tolerances against industry standards and IT grades
- **Visual Tolerance Representation**: Clean, intuitive charts and comparisons
- **Industry Benchmarks**: Compare against LEGO, Swiss watchmaking, and precision machining standards

## Usage

Simply open `index.html` in any modern web browser. No installation or setup required.

### Tools Included:

1. **Bilateral Tolerance**: Enter nominal dimensions and plus/minus tolerances to see the acceptable range
2. **Stack-up Analysis**: Add multiple component tolerances to calculate total assembly variation
3. **Precision Grade**: Input your tolerance requirements to determine the manufacturing precision level needed

## Design Philosophy

PrecisionCraft makes complex manufacturing concepts accessible through clean visualization and practical calculations. Whether you're a hobbyist maker or professional engineer, the tool helps you understand and apply precision manufacturing principles.

The interface draws inspiration from precision craftsmanship itself - clean, purposeful, and built with attention to detail.
